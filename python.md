# 🐍 Python – Key Concepts & Overview

## 📌 What is Python?

Python is a **high-level, interpreted programming language** known for its simplicity and readability. It is widely used for **web development, data science, AI/ML, automation, and scripting**.

---

## 🔑 Key Features of Python

* Easy to learn and read (simple syntax)
* Interpreted language (no compilation required)
* Dynamically typed (no need to declare variable types)
* Cross-platform (runs on Windows, Linux, Mac)
* Large standard library
* Supports multiple programming paradigms:

  * Object-Oriented
  * Procedural
  * Functional

---

## ⚙️ How Python Works (Interpreter-Based)

Python uses an **interpreter** instead of a compiler.

### 🔄 Execution Process:

1. You write code in `.py` file
2. Python converts it into **bytecode** (`.pyc`)
3. Python Virtual Machine (PVM) executes the bytecode

### ✅ Advantages of Interpreter:

* Faster development (no compile step)
* Easier debugging
* Platform-independent execution

| Type           | Interpreted       | Compiled           | Compiled + JVM                 |
| Syntax         | Simple & readable | Complex            | Moderate                       |
| Speed          | Slower            | Very Fast          | Fast                           |
